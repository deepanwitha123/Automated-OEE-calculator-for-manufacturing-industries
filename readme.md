🏭 Automated OEE Analytics Dashboard

Overview
This project is an interactive web application designed to automate the calculation and reporting of Overall Equipment Effectiveness (OEE) for industrial manufacturing. Instead of manually crunching numbers, facility managers can upload daily shift data to instantly visualize key performance indicators, simulate downtime reductions, and dispatch formatted reports via email.

Core Features:

Automated Metrics: Instantly calculates Availability, Performance, and Quality from raw CSV data.

Interactive What-If Simulator: Allows users to simulate how reducing specific downtimes impacts the final OEE score.

Root Cause Analysis: Generates Pareto charts to identify the top reasons for production loss.

1-Click Reporting: Uses SMTP protocol to generate and email a formatted HTML summary directly from the dashboard.

Tech Stack: Python | Streamlit | Pandas | Plotly
